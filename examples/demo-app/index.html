Analytics demo

This page uses the web SDK to send events to the Capture API. Events are batched (every 5 events or 3 seconds) and sent to POST /capture.

What gets captured

  • $pageview — On load (path, url, title).
  • $autocapture_click — Every click anywhere (tag, text, href).
  • feature_click — When you click “I clicked Feature” (custom).
  • signup_click — When you click “Sign up” (custom).

So clicking a button sends both the custom event and an autocapture click. “Flush queue now” sends any queued events immediately.
